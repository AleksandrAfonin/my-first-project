В рамках данной постановки оба решения абсолютно правильные.
Второй вариант менее гибкий, с ПлавающимиЖивотными,
потому что если будет добавлено еще одно свойство
(например, УмеющиеЗалазитьНаДеревья, то придется делать комбинацию из 4-х классов),
а если появится и третье свойство, то тогда будет 8 вариантов.
